# Release Checklist

## Build and Test
- shared preset Debug configure/build passes
- shared preset Release configure/build passes
- `ctest` passes in Debug and Release

## Packaging
- portable deploy folder generated by `deploy_release.bat`
- portable folder contains exe, Qt runtime, data, helper docs, packaging notices
- WiX MSI stage either passes or is explicitly marked blocked

## Runtime Smoke
- app starts and reaches main window
- live connect/disconnect
- logging start/stop/save/discard
- replay load/play/pause/seek
- issue navigation
- timing/value/alarm coherence
- recent-window graph and full-range overview graph

## Release Metadata
- build metadata matches intended baseline
- session log is emitted
- SBOM placeholder or generated SBOM is present
- third-party notices are present
- known issues captured
